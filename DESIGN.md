## Project Brief

重构现有 Local DB 中文仪表盘 UI。项目是本地优先的 PostgreSQL 知识与信号工作台，React 18 + Vite + Tailwind CSS v4 + shadcn/radix-nova。核心导航包括工作台总览、笔记库、AI 信号流、开源项目雷达、数据表、只读 SQL；主要用户是本人及可信局域网中的技术用户。保留现有信息架构、Docker Compose 运行语境、只读浏览和 CLI 写入边界；采用友好型知识平台方向，强化快速扫描、内容检索、知识阅读、数据状态、表格可读性和详情阅读。桌面优先但需响应式，中文界面，中高信息密度，保持技术专业感，避免过度柔软、泛化企业后台和过度霓虹赛博感。
